Countdown

Countdown is a JavaScript plugin to easily create countdowns and counters.

Manual starting Countdowns

Absolute values

Click to animate the Countdown to 500,1000 or 1500.

{% highlight javascript %} $('a').on('click', function(){ $('.countdown').countdown('animateTo', 0); }); {% endhighlight %}

Relative values

Click to add 1234 to or substract 1234 from the countdown.

{% highlight javascript %} $('a').on('click', function(){ $('.countdown').countdown('animateTo', '+1234'); }); {% endhighlight %}

Countdown with callback

{% highlight javascript %} $('.countdown').countdown().on('countdown:finished', function(event, data){ var _this = this; setTimeout(function(){ // Pause for 1 second and then revert to previous value $(_this).countdown('animateTo', data.old_value); }, 1000); }); {% endhighlight %}

Countdown with custom display

Customizing the looks of Countdown is very easy, just use CSS.

{% highlight css %} .countdown{ height: 70px; } .countdown .digit{ height: 70px; width: 70px; font-family: 'Courier New', Courier; font-size: 70px; line-height: 74px; border-radius: 10px; background: #a90329; background: -webkit-linear-gradient(top, #a90329 0%, #a90329 53%, #8f0222 53%, #6d0019 100%); } {% endhighlight %}
