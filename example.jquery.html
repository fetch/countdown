Counter

Counter is a JavaScript plugin to easily create counters without images. For more information (including usage, explanation and faqs), check out the project on GitHub.

Autostarting Counters

Counters can be any size. Options can be set through data-attributes or by an options object passed to the constructor.

With data-attributes:

<div class="counter" data-digits="20" data-duration="2000000" data-animate-to="1000000"></div>

<script>
$('.counter').counter();
</script>

With object in constructor:

<div class="counter"></div>

<script>
$('.counter').counter({
    digits: 20,
    duration: 2000000
}).counter('animateTo', 1000000);
</script>

Manual starting Counters

Counters can be initiated with a number and target number.

Click to update the Counter to 0 or 2000

Code:

$('a').on('click', function(){
    $('.counter').counter('animateTo', 0);
});

Click to add 1234 to or substract 1234 from the counter

Code:

$('a').on('click', function(){
    $('.counter').counter('animateTo', '+1234');
});

Counter with callback

Counters can be initiated with a number and target number.

Code:

$('.counter').counter().on('counter:finished', function(event, data){
    var _this = this;
    setTimeout(function(){
        // Pause for 1 second and then revert to previous value
        $(_this).counter('animateTo', data.old_value);
    }, 1000);
});

Counter with custom display

Customizing the looks of Counter is very easy, just use CSS.

Code:

<div class="counter custom">
   <div class="overlay"></div>
</div>
<style>
.counter .digit{
    border-radius: 3px;
    background: #990000;
}

.counter .overlay{
    height: 43px;
    width: 263px;
    position: absolute;
    top: 0;
    left: 0;
    background-image: -webkit-gradient(linear, left bottom, left top,
        color-stop( 0, rgba(255, 255, 255, 0)),
        color-stop( .5, rgba(255, 255, 255, .4)),
        color-stop( 1, rgba(255, 255, 255, 0))
    );
}
</style>

Change / Update Events

  • Counter Finished

    $('.counter').counter().on('counter:finished', function(event, data){ … } );

  • Counter Stopped

    $('.counter').counter().on('counter:stopped', function(event, data){ … } );

Setup (for jQuery)

Using Counter is easy as can be.

  1. Download the plugin and copy the counter files to your app.
  2. Activate the plugin on any div of your choice: $('.counter').counter()
  3. Modify counter.css to change the look of your counter.
